Demonstração de uso da aplicação:

```markdown
# Como Usar

1. Preencha os campos com suas informações básicas (nome, CPF, endereço, redes sociais).
2. Faça upload de uma imagem/documento para extração de texto.
3. Digite um breve resumo sobre sua personalidade.
4. O sistema validará se o texto está relacionado a e-sports usando IA.
5. O resumo dos dados será mostrado ao final.

✅ Se o conteúdo for reconhecido como fã de e-sports, uma validação positiva será exibida.
❌ Caso contrário, será mostrado que o conteúdo é genérico.
